A 2D, side-scrolling, obstacle dodge game made in Unity and C# with course material from Gamedev.tv's Unity 3D course. 

Using W to thrust and A & D to turn, the player must navigate each of the three levels from the launch pad to the landing pad without touching the walls, ceilings, or floors.
